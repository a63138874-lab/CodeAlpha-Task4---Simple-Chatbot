# CodeAlpha-Task4---Simple-Chatbot
Basic Chatbot
A simple rule‑based chatbot built in Python that responds to basic user inputs like "hello", "how are you", and "bye". This project is designed to practice core programming concepts such as if-elif, functions, loops, and input/output handling.

📌 Features
1) Takes text input from the user.

2) Recognizes predefined phrases such as:

     "hello" → Replies with "Hi!"

      "how are you" → Replies with "I'm fine, thanks!"

      "bye" → Replies with "Goodbye!" and exits the chat

3) Uses a loop to keep the conversation going until the user says goodbye.

4) Simple and easy to understand code structure.


🧩 Key Concepts Used
1. if-elif conditions for matching user messages

2. Functions to separate logic and produce replies

3. while loop for continuous conversation

4. input() and print() for user interaction
